Интегрированные типов
=============
Язык `Python` по умолчанию поддерживает вычисления с бесконечной точностью целых чисел, и можно использовать возможности `Python` для вычислений целых чисел вместо `ext-bcmath`.

Создание
-------
Используйте функцию `PyCore::int()` для создания числа, передавая в нее целое число, плавающую точку или строку для инициализации.

```php
$i1 = PyCore::int(12345678);
$i2 = PyCore::int('1234567890123456789012345678901234567890');
$i3 = PyCore::int(12345678.03);
```

Операции
---------
Цели также являются объектами `PyObject`, и можно использовать встроенные методы для выполнения операций.

```php
$i = PyCore::int(12345435);
var_dump(strval($i->__pow__(3)));
var_dump(strval($i->__add__(4)));
```

Выведет `1881564851360655187875`, и поскольку результат превышает максимальную точность `64 бит`, результат автоматически конвертируется в тип строки.
