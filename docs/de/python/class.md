# Klasse Operations


## Class laden

```python
cls = phpy.Class("class_name")
```


## Lesen der Klassen-Statischen Eigenschaft
```python
value = cls.get("name")
```


## Festlegen der Klassen-Statischen Eigenschaft
```python
cls.set("name", value)
```

## Erstellung eines Instanzes
```python
object = cls.new(arg1, arg2, arg3, ...)
```
